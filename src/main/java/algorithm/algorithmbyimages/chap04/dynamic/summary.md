# Dynamic Programming

~~~

이미 계산된 결과를 메모리에 저장한 후 다시 계산하지 않도록 만드는 것이 포인트

- top-down
- bottom-up

동적 계획법의 필요 조건은
1. 최적 부분 구조
2. 반복되는 부분 문제

점화식을 만들어 풀어야 한당
점화식이란 인접한 항들 사이의 관계식을 의미
피보나치 수열에서 An = An-1 + An-2 와 같은 것

단순 재귀로 피보나치 구현할 시 지수 시간 복잡도를 가진당

Memoization
